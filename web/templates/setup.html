2
获取 Telegram API 凭证
需要在 Telegram 官方申请 API ID 和 API Hash。这是使用 Telegram 服务的必要凭证。
📝 申请步骤：
  1. 访问 https://my.telegram.org
  2. 使用你的 Telegram 账号登录（手机号 + 验证码）
  3. 点击 "API development tools"
  4. 点击 "Create new application"
  5. 填写应用信息（名称和描述可以随意填写）
  6. 提交后获得 App api_id 和 App api_hash
⚠️ 重要提示：
  • api_id 是一个数字
  • api_hash 是一个 32 位的字符串
  • 请妥善保管，不要泄露给他人
← 上一步 保存并下一步 →
